# Searchable Autocomplete (Vite + React + MUI)

This is a standalone searchable autocomplete component with Persian/Arabic text normalization, built using React and Material-UI (MUI), ready to run on Vite.

## 📦 Installation

```bash
npm install
npm run dev
```

## ✨ Features

- Search by Persian/Arabic with normalization (ی / ك / ة / ... handled)
- Works with `shahrDesc` field
- Supports exact and partial matching

## 🚀 Demo

Clone and run locally:

```bash
git clone https://github.com/your-username/searchable-autocomplete.git
cd searchable-autocomplete
npm install
npm run dev
```
